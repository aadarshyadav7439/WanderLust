🏡 WanderLust

WanderLust is an Airbnb-inspired web application where users can discover, create, edit, and manage travel stays and vacation rentals.

This project is currently under development as I continue building new features and improving the application.

✨ Current Features

* Browse all listings
* View listing details
* Create new listings
* Edit listings
* Delete listings

🚧 Coming Soon

* User authentication
* Reviews & ratings
* Image uploads
* Booking system
* Interactive maps
* Search & filters

💻 Built With

* Node.js
* Express.js
* MongoDB
* Mongoose
* EJS
* JavaScript
* HTML & CSS

🚀 Run Locally

npm install
npm start

Visit:

http://localhost:8080/listings

Inspired by Airbnb and built for learning full-stack web development.
